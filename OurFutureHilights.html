Our Future Hilights

อนาคตที่พวกเราหวังไว้

อนาคตของ "เตย"

อาชีพหลังจบการศึกษา : Web developer เก็บเกี่ยวประสบการณ์เยอะๆและความรู้ให้มาก เมื่อมากพอแล้วอยากทำในตําแหน่ง PM
ทักษะที่อยากพัฒนา : การบริหารธุรกิจ การพูดคุย กฎหมาย ภาษา
อยากทำงานในบริษัทอะไร : ที่สุดในใจคือ Google SCG หรือที่ไหนก็ได้ที่อยู่แล้วเรามีความสุข
ในอนาคตอยากทำอะไรเป็นพิเศษ : ทำธุรกิจส่วนตัว มีเวลาให้ครอบครัว ไปเที่ยว
ภาพตัวเองในอนาคตเป็นอย่างไร : ประสบความสำเร็จ ในหน้าที่การงาน
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : ครอบครัว ความสุขของตัวเอง

อนาคตของ "ฮิม"

อาชีพหลังจบการศึกษา : อยากจะประกอบอาชีพทางด้านsoftware engineer
ทักษะที่อยากพัฒนา : การพูด ภาษา ความสามารถในด้านต่างๆ
อยากทำงานในบริษัทอะไร : อยากทำบริษัทที่เกี่ยวข้องกับซอฟแวโดยตรง
ในอนาคตอยากทำอะไรเป็นพิเศษ : เป็นเจ้าของอสังหาริมทรัพย์
ภาพตัวเองในอนาคตเป็นอย่างไร : สามารถดูแลครอบครัวได้
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : ครอบครัวและเพื่อนๆ

อนาคตของ "สายป่าน"

อาชีพหลังจบการศึกษา : เริ่มต้นหลังเรียนจบอาจจะเริ่มทำ Developer ให้มีประสบการทำงาน แต่อยากจะเน้นไปทางด้าน Analyst
ทักษะที่อยากพัฒนา : การพูด การเข้าสังคม ตอนนี้เริ่มอยากเรียนภาษาที่ 3 เพิ่มขึ้นไว้และทักษะที่เกี่ยวกับสาขาวิชา การใช้โปรแกรม การพัฒนาซอฟแวร์
อยากทำงานในบริษัทอะไร : หลังเรียนจบสามารถทำงานในบริษัทไหนก็ได้ให้มีประสบการณ์ก่อน หลังจากมีประสบการณ์การทำงานมากพอค่อยหาโอกาสการเติบโตทางการงาน
ในอนาคตอยากทำอะไรเป็นพิเศษ : ถ้ามีโอกาสอยากย้ายไปอยู่ต่างประเทศ
ภาพตัวเองในอนาคตเป็นอย่างไร : เรียนจบ อยู่กับครอบครัว มีความสุขกับชีวิตที่เป็นอยู่
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : ครอบครัวและตนเอง

อนาคตของ "แป้ง"

อาชีพหลังจบการศึกษา : นักทดสอบซอฟต์แวร์หรือนักวิเคราะห์ข้อมูลการตลาดและธุรกิจ
ทักษะที่อยากพัฒนา : อยากพัฒนาเกี่ยวกับภาษาอังกฤษและภาษาที่3 รวมถึงการพูด
อยากทำงานในบริษัทอะไร : บริษัทที่ต้อนรับเราและเราอยากจะทำงานด้วย
ในอนาคตอยากทำอะไรเป็นพิเศษ : อยากเลี้ยงแมวและไปอยู่ต่างประเทศ
ภาพตัวเองในอนาคตเป็นอย่างไร : เป็นคนธรรมดาที่มีความสุขกับสิ่งเล็กๆน้อยๆ ได้นอนเยอะๆ
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : ตนเองและครอบครัว

อนาคตของ "ตุ้ย"

อาชีพหลังจบการศึกษา : อะไรก็ได้ที่เกี่ยวกับ developer
ทักษะที่อยากพัฒนา : ทักษะการสื่อสารเเละเข้าสังคม
อยากทำงานในบริษัทอะไร : บริษัทเล็กๆที่มีพนักงานไม่กี่คน
ในอนาคตอยากทำอะไรเป็นพิเศษ : ถ้ามั่นคงเเล้วจะออกจากงานไปใช้ชีวิตเรียบง่าย
ภาพตัวเองในอนาคตเป็นอย่างไร : มีความสุขที่มีชีวิต
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : เพื่อความสุขของตนเอง

อนาคตของ "หงส์"

อาชีพหลังจบการศึกษา : Developer/Soft-eng ไม่ก็เกี่ยวกับ Fintech
ทักษะที่อยากพัฒนา : ความรู้ด้านการเงิน ภาษา กฎหมาย และการเขียนโปรแกรม
อยากทำงานในบริษัทอะไร : KBTG, Agoda
ในอนาคตอยากทำอะไรเป็นพิเศษ : ถ้าทุกอย่างลงตัวแล้ว อยากพาครอบครัวไปเที่ยว และทำอะไรใหม่ๆ
ภาพตัวเองในอนาคตเป็นอย่างไร : มีความสุขกับสิ่งที่ทำ มีความก้าวหน้า
แรงบันดาลใจที่ตั้งเป้าหมายนี้ : ครอบครัวและตนเอง
